# Local Font Viewer (Japan Edition) - 完成報告書

## 概要
俺様とお前で作った、最強のローカルフォント閲覧アプリだ！
PCに入ってるフォントを全部読み込んで、一覧表示・検索・プレビューができる。
特に「日本語対応フォント」の判定ロジックには、俺様たちの血と汗と涙（とデバッグログ）が詰まってるぜ！

## 主な機能

### 1. 爆速フォント一覧表示
- ブラウザの `queryLocalFonts` API を使って、PC内のフォントを瞬時にリストアップする。
- 数百個あってもグリッド表示で見やすいぜ。

### 2. 日本語フォント・フィルター (The 3-Stage Rocket Logic)
これが今回の目玉だ！
ただの幅チェックだと「Arial」みたいなやつに騙されるから、以下の**鉄壁の二段構え**で判定してる。

1.  **名前で顔パス (Heuristic)**
    - 「Mincho」「Gothic」「Anito」「Hana」みたいな名前のやつは、問答無用で合格！
2.  **見た目一致チェック (Pixel Consistency)**
    - 名前で漏れたやつは、裏でこっそり検査する。
    - 「ゴシックの隣」と「明朝の隣」で、文字（『あ』）の見た目が変わるかどうかをピクセル単位で比較。
    - **「周りに流されず、自分を貫くやつ（見た目が変わらない）」**だけを合格させる！
    - これで、システム標準と同じ幅のフォントでも正確に見分けられるようになった。

### 3. "Live" プレビュー & カスタマイズ
- 好きな言葉（「愛してるぜー！」とか）を入力すれば、全カードにリアルタイムで反映される。
- 気に入ったフォントはクリック一発で詳細確認。
- **「Use for App UI」**ボタンで、このアプリ自体のフォントもその場で変えられる！

### 4. 爆速起動ランチャー (`launch.bat`)
- ChromeだとAPIが動かねぇことがあるからな。
- このファイルをダブルクリックするだけで、勝手に **Edge** が立ち上がってアプリが開く！

## 使い方はシンプルだ

1. `launch.bat` をダブルクリックする。
2. Edgeが立ち上がったら、「フォントの閲覧を許可しますか？」って聞かれるから「許可」する。
3. あとは好きなだけフォントを眺めてニヤニヤしろ！
